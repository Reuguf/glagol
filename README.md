Установка:
1. Запусти install.cmd
2. Закрой и заново открой cmd
3. Проверь:
   glagol версия
   glagol "%LOCALAPPDATA%\Glagol\examples\hello.гл"

Сборка автономного установщика:
1. Запусти BUILD_EXE.cmd
2. Забери release\GlagolSetup_0_3.zip
3. Этот zip можно скинуть другу. Python другу не нужен, если внутри есть glagol.exe.

Команды:
glagol файл.гл
glagol файл.гл --python
glagol файл.гл --безопасно
glagol проверить файл.гл
glagol создать консоль
glagol создать окно
glagol создать графика
glagol версия
